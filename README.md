# Quendor

The aim of Quendor is to create a [Glulx](https://www.eblong.com/zarf/glulx/) implementation in JavaScript. There is already such an implementation known as [Quixe](https://github.com/erkyrath/quixe). My goal here is not to be entirely unique nor to create a competing interpreter.

Instead my goal is to create a viable implementation to (1) learn how to create such an interpreter and (2) to level up my JavaScript skills.

My project documentation will get more comprehensive as my project evolves.
